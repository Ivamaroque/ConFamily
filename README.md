## 📋 Sobre

O **ConFamily** é um sistema de controle financeiro moderno e intuitivo, desenvolvido para ajudar você e sua família a gerenciar despesas de forma inteligente e colaborativa.

Com uma interface amigável e recursos inovadores, o ConFamily permite que você tenha total controle sobre seus gastos, facilitando a divisão de despesas e o planejamento financeiro familiar.

## ✨ Funcionalidades

### 📝 Inserção Manual de Dados
- Adicione seus gastos manualmente de forma rápida e simples
- Interface intuitiva para registro de despesas diárias

### 🤖 Importação Inteligente com IA
- **Integração com GPT API**: Upload automático de faturas de cartão
- Processamento inteligente de dados usando IA
- Extração automática de informações das faturas

### ✅ Validação e Personalização
- **Revisão de Dados**: Avalie e confirme as informações importadas
- **Categorização Flexível**: Escolha o tipo de gasto para cada despesa
- **Divisão Proporcional**: Defina quanto de um valor pertence a cada membro da família
- **Gestão Compartilhada**: Identifique facilmente as parcelas de cada pessoa

### 📊 Controle Completo
- Visualize seus gastos por categoria
- Acompanhe despesas individuais e compartilhadas
- Relatórios detalhados de gastos familiares

## 🚀 Tecnologias

- **Inteligência Artificial**: OpenAI GPT API para processamento de faturas
- **Interface Moderna**: Design responsivo e intuitivo
- **Segurança**: Proteção de dados financeiros sensíveis

## 💡 Como Usar

### Método 1: Inserção Manual
1. Acesse a interface de cadastro
2. Preencha os dados da despesa (valor, categoria, data)
3. Defina a divisão entre os membros da família
4. Confirme e salve

### Método 2: Importação Automática
1. Faça upload da fatura do seu cartão de crédito
2. A IA processará e extrairá as informações automaticamente
3. Revise os dados importados
4. Ajuste categorias e divisões conforme necessário
5. Aprove e salve os lançamentos

## 🎯 Diferenciais

- ✅ **Automação Inteligente**: Economize tempo com importação via IA
- ✅ **Colaborativo**: Perfeito para controle financeiro familiar
- ✅ **Flexível**: Adapte-se a diferentes modelos de divisão de gastos
- ✅ **Transparente**: Visualize claramente quem gastou o quê
